# Project Brief

*   **Core Requirements:** 
    *   Log ski days with details: date, resort, skis, companions, activity (training, powder, etc.), photo.
    *   View logged ski days on a calendar (monthly/weekly).
    *   View logged ski days on a map.
    *   Filter ski days by equipment, people, etc.
    *   Backend API for data storage and retrieval.
*   **Goals:** Create an app for skiers to easily log and review their skiing activities visually (calendar, map) and through filtered lists.
